Background Image

Background Image

Das Hintergrundbild benötigt ebenso ein alt Attribute. Durch leer lassen das alt Attributes wird dem SR vermittelt, dass es sich um ein dekoratives Element handelt. Alternativ geht auch ein aria-hidden. 

            
<img 
  id="background" 
  src="src/pexels-pixabay-531880.jpg" 
  alt="Blick auf Stadt durch verregnetes Fenster" 
/>

Standard Image

Jedes Bild benötigt ein alt-Attribute, dieses wird vom SR vorgelesen. Es wird aber auch angsagt dass es sich um ein Bild handelt. Die Elementbezeichnug muss niemals im label- oder alt-Attribute Attribute notiert werden. 

            
<img 
  src="https://cataas.com/cat/funny?type=square" 
  alt="Bild von lustiger Katze"
/>
            
          
             
<img 
  src="https://cataas.com/cat/funny?type=square" 
  alt="lustige Katze"
/>

Nachgebauter Button

Der native Button bringt Funktionalität und Styling mit. Zu beachten sind auch hover, focus und active styles. 

      
  <!-- simple button -->
  <button onclick="alert('Hallo!')">HTML 5 Standard Button</button>
DIV mit Klick Event
 
              
  
  <!-- simple div screenreader will initially read this but that's all -->
  <div onclick="alert('Hallo!')">DIV mit Klick Event</div>
DIV mit Role Button 
              
  <style>
  div[role="button"] {
    border: 1px solid black;
    border-radius: 5px;
    padding: 0.5rem;
    margin: 1rem;
    cursor: pointer;
  
    &:hover {
      background-color: #eee;
    }
  }
  </style>
  <!-- screenreader will know its clickable -->
  <div role="button" onclick="alert('Hallo!')">
    DIV mit Role Button
  </div>
DIV mit Tab Index 
              
  
  <!-- keyboard kann button fokussieren 
    wenn ein Screenreader läuft, löst Return und Space das onclick Event
    aus, wenn nicht dann nicht -->
  <div tabindex="0" role="button" onclick="alert('Hallo!')">
    DIV mit Tab Index
  </div>
DIV mit on keyup Event 
              
  
  <!-- alert wird bei jedem Tastatur Druck ausgelöst -->
  <div
    tabindex="0"
    role="button"
    onclick="alert('Hallo!')"
    onkeyup="alert('Hallo!')"
  >
    DIV mit onkeyup Event
  </div>
DIV mit Eventlistener 
              
  <!-- alert wird mit Return und Space ausgelöst -->
  <div
    aria-label="Alert the word Hallo"
    tabindex="0"
    role="button"
    onclick="alert('Hallo!')"
    id="btn"
  >
    DIV mit eventlistener
  </div>
  </div>
  <script>
  document.getElementById("btn").addEventListener("keyup", test);
  function test(event) {
    if (event.key === "Enter" || event.key === " ") {
      event.preventDefault();
      alert("Hallo!");
    }
  }
  </script>
